# FNP Sales Analysis Dashboard

This project presents a sales analysis of FNP (Ferns N Petals) using Excel. The dashboard highlights key business metrics, revenue patterns, and customer behavior insights. It helps understand sales performance across categories, products, cities, occasions, and time periods.

### **Key Features**

* Total orders, total revenue, average spending, and delivery time KPIs
* Revenue analysis by **occasions**, **categories**, and **months**
* Hourly order trend showing customer activity
* Top 5 products and top 10 cities analysis
* Filters for date and occasion selection
* Interactive charts for clear business insights

### **Tools Used**

* Microsoft Excel
* Pivot Tables
* Charts and Slicers

### **Objective**

To analyze sales data and identify patterns that support better decision-making for product performance, peak sales times, and customer preferences.
